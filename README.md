## ⚔️ Features

### Weapons
- **5 weapon types** — Pistol, Rifle, Shotgun, Sniper Rifle, SMG
- **Full attachment system** — Red Dot, Holo, Laser, Flashlight, 4x/8x Scopes
- **3D models** with smooth ADS transitions and scope overlays
- **Realistic mechanics** — Reload (R key), aim down sights, bullet penetration, bleeding effect

### Production Chain
- **Electric Blast Furnace** with coil tiers (Kanthal 800°C / Nichrome 1200°C / Tungsten 2000°C)
- **Coke Oven** multiblock → Coal Coke
- **Lathe, Heat Treatment Furnace, Assembly Bench** for precision machining

### Energy System
- **Coal Generator** (80 FE/tick) → **Fuel Generator** (up to 200 FE/tick)
- **Copper Wire** network with wrench configuration
- **Energy Ports** for multiblock power connections

### Oil Refining
- Butane, Gasoline, Naphtha, Kerosene, Diesel, Heavy Gas Oil, Residual Fuel Oil
- **Fluid Pipes** and **Fluid Ports** with input/output modes and fluid targeting

### Chemistry
- **Chemical Mixer** — Acids, PVC, Synthetic Rubber
- **Acid Bath** — Etched metals
- **Electrolysis Machine** — Industrial gases
- **Water Pump** — Automated water supply

### Combat & Defense
- **Bulletproof Vest** — 60% bullet damage reduction
- **Night Vision Goggles** — FE powered
- **Auto Turret** — Automatic mob defense
- **Bleeding mechanic** + **Bandages** + **Medikits**
- **Grenades, Smoke Grenades, Land Mines, Sea Mines**

### World Generation
- **8 new ores** — Sulfur, Saltpeter, Tungsten, Fluorite, Uranium, Chromium, Nickel, Bauxite
- **Military Bunkers** — Rare underground structures with weapon loot
- **Oil deposits** underground

### Automation
- **Item Pipes** with per-side EXTRACT/INSERT modes
- **Fluid Pipes** with wrench configuration
- **Wrench** tool for pipe/wire/port management

## 🏭 Quick Start Guide

### First Steps
1. Mine **Iron Ore** and smelt into **Steel Ingots** (furnace)
2. Build a **Metal Press** to create intermediate parts
3. Build a **Gunsmith Assembly Bench** to craft your first **Pistol**
4. Craft **Bullets** from iron nuggets + gunpowder

### Power Generation
1. Build a **Coal Generator** and connect with **Copper Wire**
2. Power your machines — **Lathe**, **Heat Treatment Furnace**, **Assembly Bench**

### Oil Production
1. Find **Oil** underground (Y=-20 to Y=20)
2. Build the **Oil Derrick** multiblock (3x3 base + 5 pillars)
3. Build the **Refinery** multiblock (5x5x6)
4. Use **Fluid Pipes** and **Fluid Ports** to connect them

### Advanced Steel
1. Build a **Coke Oven** (3x3x3) to produce **Coal Coke**
2. Build an **Electric Blast Furnace** (3x3x4) with **Kanthal Coils**
3. Process ore much more efficiently
